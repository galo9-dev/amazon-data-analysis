# 📊 Amazon Products Data Analysis

## 📌 Objective
Analyze Amazon product data to extract insights about pricing, ratings, and categories.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
Amazon products dataset including price, discount, rating, and reviews.

## 📊 Analysis Performed
- Data cleaning (prices, percentages, ratings)
- Top-rated products
- Most expensive products
- Average rating by category
- Discount analysis

## 📈 Key Insights
- Office Products category has the highest average rating
- Car & Motorbike category has the lowest rating
- High discounts do not always correlate with better ratings
