# Amazon Products Dashboard

Este proyecto es un dashboard interactivo que armé para analizar productos de Amazon usando Python.  
Me permite explorar los datos, filtrar productos y ver métricas y gráficos de forma sencilla.

## Qué hace

- Puedo filtrar productos por **categoría**, **precio máximo** y **rating mínimo**.  
- Veo métricas como:
  - Rating promedio
  - Precio promedio
  - Cantidad de productos
- Muestra los **Top 10 productos** según rating y cantidad de reviews.  
- Graficar:
  - Rating promedio por categoría
  - Precio promedio por categoría  

Es básicamente un proyecto para practicar **Python, Pandas y Streamlit**, pero también queda lindo como portfolio.

## Tecnologías

- Python  
- Pandas  
- Matplotlib  
- Streamlit  

## Cómo usarlo

1. Clonar el repo:
```bash
git clone <TU_REPO_URL>
```

2.Instalar librerias necesarias:
```bash
pip install pandas matplotlib streamlit
```

3. Ejectuar dashboard:
```bash
streamlit run app.py
```
